# ktk

An alias script to compile and execute kotlin with ease !

Steps to follow


1. Download ktk.sh and save at your desired location

2. open .bashrc and add an alias to ktk.sh
   example: 
     alias kt="/home/aakash/Desktop/ktk.sh"

3. reload bash & done

Syntax

 
Compile & Run:
  kt file.kt
  
Kotlin shell:
  kt
